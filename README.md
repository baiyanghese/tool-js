# tool-js
javascript tool function
本函数包不依赖任何框架，完全采用原生js实现，可以放心使用，总大小为9kb，压缩后为4kb,非常适合手机端使用。

### 函数列表
``` javascript

TOOL.valid_email #验证邮箱
TOOL.valid_phone #验证手机号
TOOL.valid_idcard #验证身份证号
TOOL.valid_url #验证网址
TOOL.valid_wxbrowser #判断是否微信浏览器
TOOL.alert_tips ＃弹窗提示
TOOL.remove_html_tag ＃移除含有html标签的字符串里的html
TOOL.html_to_escape ＃html标签转为转义字符
TOOL.start_with ＃判断是否以某个字符开始
TOOL.end_with ＃判断是否以某个字符结束
TOOL.get_current_page_all_url ＃抓取当前页面所有url，包括图片地址
TOOL.get_param ＃根据参数名获取当前url的参数值
TOOL.unique_id ＃获取随机数
TOOL.get_pageview_height ＃获取页面的可视区域高度
TOOL.get_pageview_width ＃获取页面的可视区域宽度
TOOL.get_scroll_xy ＃获取网页被卷取的位置
TOOL.is_mobile_agent ＃判断是否为移动设备
TOOL.is_apple_mobile_device ＃判断是否为苹果移动设备
TOOL.is_android_mobile_device ＃判断是否为安卓移动设备
TOOL.set_cookie  ＃设置cookie
TOOL.get_cookie  ＃读取cookie
TOOL.del_cookie  ＃删除cookie

```
